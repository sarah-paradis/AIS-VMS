Data and code needed to reproduce the study “Paradis, S., Sala, J., Durán, R., Dordal, J., Montero, I., Reynés, A., Selvaggi., M. Best practices in quantifying fishing effort using VMS and AIS data: a case study of the NW Mediterranean bottom trawling fleet”


Note that for confidentiality issues, all vessel identifiers were removed and fake identifiers were given instead. In addition, vessel characteristics (e.g., length over all, gross tonnage, horsepower) were also removed to ensure confidentiality of the vessels and prevent them to be retraceable. Hence, while this code allows to reproduce the data of the aforementioned study, it does not allow the user to re-interpret the data (e.g., cross-check which vessels broadcast in both AIS and VMS and which are their characteristics).
